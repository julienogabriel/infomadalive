# Premiers contenus à entrer dans le bot

## Coupures JIRAMA (exemples à adapter)

Dès que tu as une vraie info, tape dans le bot :

```
/addcoupure Analakely | 1er Avril 2026, 8h-12h | 4h
/addcoupure Ivandry | 2 Avril 2026, 6h-10h | 4h
/addcoupure Ambohijatovo | 1er Avril 2026, 14h-18h | 4h
```

Sources pour les coupures :
- Page Facebook JIRAMA officielle
- Groupes Facebook de quartier
- Bouche à oreille (vérifie avant de publier)

## Premiers broadcasts à envoyer

### Broadcast 1 - Bienvenue
```
/broadcast Bienvenue sur Info Mada Live ! Tu recevras ici chaque jour : les coupures JIRAMA, les bons plans prix, et les offres d'emploi. Partage avec tes amis !
```

### Broadcast 2 - Engagement
```
/broadcast Sondage rapide : quelle info tu veux recevoir en PRIORITÉ ? Réponds par le numéro : 1) Coupures JIRAMA 2) Bons plans prix 3) Offres d'emploi 4) Tout !
```

### Broadcast 3 - Viralité
```
/broadcast On est déjà [X] abonnés ! Objectif : 1000 cette semaine. Partage le bot avec 3 amis et aide ta communauté à rester informée !
```

1. **Coupures JIRAMA** : Page Facebook JIRAMA, groupes quartier
